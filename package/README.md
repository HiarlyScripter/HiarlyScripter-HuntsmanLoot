---

## 🇧🇷 Resumo

O **Huntsman Loot** faz o Huntsman dropar a **Huntsman Rifle** ao ser eliminado. A arma usa o visual nativo **Hunter Gun** do próprio jogo via referência em runtime — sem empacotar assets extraídos, sem copiar assets de mods de referência, sem dependências externas além do BepInEx.

## 🇺🇸 Summary

**Huntsman Loot** makes the Huntsman drop the **Huntsman Rifle** when killed. The weapon uses the game's native **Hunter Gun** visual at runtime — no extracted assets bundled, no third-party mod assets copied, no external dependencies beyond BepInEx.

## ⚙️ Configurações / Configuration

Edite em `BepInEx/config/com.hiarlyscripter.huntsmanloot.cfg` ou use o **REPOConfig** in-game.

| Seção / Section | Chave / Key | Padrão / Default | Descrição / Description |
|---|---|---|---|
| `Drop` | `DropChance` | `100` | Chance (%) de drop — 1 a 100 / Drop chance (%) — 1 to 100 |
| `Drop` | `BerserkerOnly` | `false` | Só dropa de Huntsmans berserk / Only drops from berserk Huntsmans |
| `Drop` | `MasterClientOnly` | `true` | ⚠️ Manter true — evita drops duplicados / Keep true — prevents duplicate drops |
| `Drop` | `RandomizeAmmo` | `true` | Munição aleatória entre 1 e o máximo / Random ammo between 1 and max |

## 📋 Notas técnicas / Technical notes

- O mod usa o visual nativo **Hunter Gun** do jogo via referência em runtime — não redistribui assets extraídos do jogo.
- The mod uses the game's native **Hunter Gun** visual at runtime reference — does not redistribute extracted game assets.
- Não copia nem modifica assets de mods de referência / Does not copy or modify assets from reference mods.
- A base funcional é a shotgun nativa do jogo (`item_gun_shotgun`) com o visual do Hunter Gun sobreposto / The functional base is the game's native shotgun with the Hunter Gun visual overlaid.

---

## ⚠️ Limitações conhecidas / Known limitations

- Por ser uma arma longa usando a base funcional da shotgun nativa, pode haver clipping visual leve em paredes ou cantos. Não afeta o gameplay.
- Being a long weapon using the native shotgun's functional base, there may be minor visual clipping against walls or corners. Does not affect gameplay.
